# Search Agent Lab

- This repository owns the Search Agent Study Group's runnable learner setup, small starter code, notebooks, tests, and later assignments.
- The sibling Search Agent Study Group repository owns planning, weekly materials, HTML slides, GitHub Pages, and organization. Do not modify it from this repository.
- Keep onboarding small and reproducible. Add a dependency manifest and lock only after the exact environment has passed this repository's setup check.
- Keep credentials local. Never commit .env files, API keys, service-account files, or captured request/response data that could be sensitive.
- Keep committed notebooks output-free. Commit source and deterministic checks, not captured interactive or live-run outputs.
- Start with the Gemini API-key path. Treat Vertex AI / Google Cloud as a later, explicit option.
- Do not copy the upstream shop_agent.ipynb into this repository. Run that instructor specimen later from a pinned, separate upstream checkout.
- White-box teaching may show safe runtime interfaces and redacted events; it must never expose private reasoning, raw traces, or secrets.
- personalized-shopping is out of scope until it is explicitly opened.
